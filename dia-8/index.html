Introdução

O PHP (um acrônimo recursivo para PHP: Hypertext Preprocessor) é uma linguagem de script open source de uso geral, muito utilizada, e especialmente adequada para o desenvolvimento web e que pode ser embutida dentro do HTM

Criado por Rasmus Lerdorf em 1995, o PHP tem a produção de sua implementação principal, referência formal da linguagem, mantida por uma organização chamada The PHP Group. O PHP é software livre, licenciado sob a PHP License, uma licença incompatível com a GNU General Public License (GPL) devido a restrições no uso do termo PHP.

Como é uma linguagem que trabalha do lado do servidor você pode manipular banco de dados, arquivos, enviar e-mail e muito mais.

Última versão: 8.0.1

Porque
  • Trabalha do lado do servidor
  • Tipagem dinámica
  • Linguagem de fácil sintaxe
  • Vários recursos avançados
  • Uma linguagem de vários paradigmas
  • Prodedural
  • Funcional
  • Orientação a objetos
Variáveis

Variáveis são como gavetas, servem para armazenar coisa, essas coisa podem ser de vários tipos como shot, camisa e meia. Trazendo para computação nas variáveis podemos guardas nomes(strings), números inteiros(int), números reais(float) entre outros.

Em PHP para se declarar uma variável ultiliza-se o $ antes do nome, sempre no final do código não esqueça de ultilizar ';' ponto e virgula.

$nome = "Leandro";
$num1 = 10;
$num2 = 50.40;
Operadores

Operadores matemáticos são ultilizados para fazerem operações mátematicas como; adição, subtração, divisão e multiplicação.

Para fazer uma operação matemática

$soma = $num1 + $num2; // adição

$sub = $num1 - $num2; // subtração

$div = $num1 / $num2; // divisão

$mult = $num1 * $num2; // multiplicação
Condicional

Condicional, é muito importante em programação. É ultilizar para romper o fluxo linear do algoritmo, que é execultar uma linha apos a outra, por algums motivo você quer que parte do seu código não seja execultado se caso aconteça.

Para criar uma condição ultilizamos a estrutura if, else if e else

if($media >= 7){
echo "Parabens voce passou";
}else if($media >= 6){
echo "Você ainda tem chance, recuperação";
}else{
echo "Ops, não foi dessa vez";
}
Repetição

Repetição é de suma importancia na programação é ultilizada quando temos código sendo repetido, então não escrevemos várias vezes algo, apenas colocamos dentro de um loop e aquela parte do código pode ser repetida 1000 vezes se escrever aquele código 1000 vezes.

Para criar um loop existe duas formar com for e while

O loop while é o enquanto

$i = 1;
while(true){
if ($i > 2){
break;
}
echo "Interação while i = $i";
$i++;
}

O loop for é um iterador

for($i=0; $i < 2; $i++){
echo "Interação for = $i";
}
Array

Arrays são um tipo de dados complexo, que funciona como uma lista, pode ter vários dados e de diversos tipos

Para fazer um array simples, usa a função array()

$numeros = array(10, 20, 30, 40, 50);

$lista = array("nome", "Leandro", "idade", 21, "Cidade", "Tucano");

Para criar uma array de mais dimencões

$matriz = array(
array("XJ6", 16000),
array("Pop100", 7000),
array("150", 8500)
);
Função

Esse é um paradigma de programação, funções são essenciais, criamos bloco de código para realizar algum procedimentos, e pode chamar onde quisermos e quando quisermos basta chamar o nome da função exemplo: gerar_Pdf()

Para criar uma função usa a palavar reservada function

function potencia($number, $n_pontecia){
for($i = 1; $i<($n_pontecia-1); $i++){
$number *= $number;
}
return $number;
}
